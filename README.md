# Pixel Studio

Pixel Studio es un editor web movil para crear texturas pixel art, iconos e items para complementos de Minecraft Bedrock.

## Alfa 0.1

Funciones actuales:

- Crear lienzos de 16x16, 32x32 y 64x64.
- Pintar pixel por pixel.
- Borrar pixeles.
- Limpiar el lienzo.
- Elegir color.
- Exportar la textura como PNG.
- Interfaz pensada primero para iPhone y Android.

## Filosofia

Pixel Studio nace como una herramienta movil primero. La prioridad es que funcione bien en celulares, sin depender de una computadora.
